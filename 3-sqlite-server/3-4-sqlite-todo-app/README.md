# 3-sqlite-server
## 3-4-sqlite-todo-app
### サーバーの実行環境
`3-sqlite-server/3-4-sqlite-todo-app/server` に `cd` した上で、以下のコマンドを実行する必要があります:
```bash
npm install
```

### 実行方法
#### サーバー
```bash
npm run start
```

#### クライアント
サーバーを実行した後に、ブラウザで `3-sqlite-server/3-4-sqlite-todo-app/client/index.html` を開いてください。
