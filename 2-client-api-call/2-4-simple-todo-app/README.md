# 2-client-api-call
## 2-4-simple-todo-app
### サーバーの実行環境
`2-client-api-call/2-4-simple-todo-app/server` に `cd` した上で、以下のコマンドを実行する必要があります:
```bash
npm install
```

### 実行方法
#### サーバー
```bash
npm run start
```

#### クライアント
サーバーを実行した後に、ブラウザで `2-client-api-call/2-4-simple-todo-app/client/index.html` を開いてください。
